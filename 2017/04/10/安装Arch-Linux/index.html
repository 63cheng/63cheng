Arch Linux安装

验证启动模式：

 ls /sys/firmware/efi/efivars

连接到因特网：

ping -c 3 archlinux.org

更新系统时间：

timedatectl set-ntp true

建立硬盘分区：

 cfdisk

为/挂载点格式化分区：

 mkfs.ext4

格式化swap分区并启用：

 mkswap
 swapon

挂载分区：

 mount /dev/sda /mnt
 mkdir /mnt/boot
 mount /dev/sda /mnt/boot

选择镜像：

vi /etc/pacman.d/mirrorlist

Server = http://mirrors.ustc.edu.cn/archlinux/repo/os/repo/os/repo/os/arch
Server = http://mirrors.tuna.tsinghua.edu.cn/archlinux/repo/os/repo/os/repo/os/arch
Server = http://mirrors.sohu.com/archlinux/repo/os/repo/os/repo/os/arch

备注：更改镜像列表后请务必使用pacman -Syy强制刷新数据库。

安装基本系统：

pacstrap -i /mnt base base-devel

生成fstab：

genfstab -U /mnt >> /mnt/etc/fstab

检查生成的fstab是否正确：

vi /mnt/etc/fstab

备注：若在运行genfstab或是之后发生错误，请勿再次运行genfstab，而是直接手动编辑fstab文件。

配置基础系统：
arch-chroot /mnt

设置Locale：

nano /etc/locale.gen

en_US.UTF-8 UTF-8
zh_CN.UTF-8 UTF-8

生成locale讯息：

locale-gen
echo en_US.UTF-8 > /etc/locale.conf

设置时区：

ln -s /usr/share/zoneinfo/Asia/Shanghai /etc/localtime


hwclock --systohc --utc

设置主机名：

echo Lenovo > /etc/hostname

设置Root密码：

passwd

启用 Intel 微指令更新

pacman -S intel-ucode

使用Uefi引导

bootctl --path=$esp install
bootctl update
nano /boot/loader/entries/arch.conf

title          Arch Linux
linux          /vmlinuz-linux
initrd         /intel-ucode.img
initrd         /initramfs-linux.img
options        root=PARTUUID=14420948-2cea-4de7-b042-40f67c618660 rw

/usr/share/systemd/bootctl 提供了参考示例文件.

exit
reboot
systemctl enable dhcpcd.service

创建用户

useradd -m -g wheel -s /bin/bash lsc
passwd  lsc
nano /etc/sudoers

安装X：

pacman -S xorg-server

安装驱动：

pacman -S alsa-utils
pacman -S xf86-input-synaptics
pacman -S xf86-video-intel

Deepin 桌面安装：

pacman -S deepin LightDM 	deepin-session-ui

安装yaourt

nano /etc/pacman.conf

[archlinuxcn]
SigLevel = Optional TrustAll
Server = https://mirrors.ustc.edu.cn/archlinuxcn/$arch

pacman -S archlinuxcn-keyring 
pacman -Sy base-devel yaourt

安装常用字体：

pacman -S ttf-dejavu wqy-microhei

安装无线网卡驱动

yaourt -S broadcom-wl
pacman -S iw wpa_supplicant dialog
systemctl disable dhcpcd.service
wifi-menu
