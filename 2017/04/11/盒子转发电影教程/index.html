盒子转发电影教程
2017-04-11|书籍

仅在ks2 ubuntu系统上尝试并成功
仅供其他类型的盒子参考

解压0day分卷资源(非0day资源可忽略)
安装解压软件unrar:

apt-get install unrar

切换到0day资源目录

cd /home/xxx/

查看目录下的文件

ls

解压压缩文件

unrar e *.rar

解压完再次查看

ls

挂载ISO文件
首先新建一个挂载点（可以随便建立）

mkdir /home/cdrom

挂载

mount -o loop -t udf *.iso /home/cdrom

查看挂载内容

ls /home/cdrom/

扫描BDInfo
由于BDInfo只有win版没有linux版，需要用mono或wine来运行
个人选择mono
安装mono

apt-get install mono-complete

可能会遇到依赖问题，只需按照提示运行

apt-get –f install

然后再进行安装。
上传或下载BDInfoCli.exe到盒子里
我把它放在了/home/下
运行BDInfoCli.exe

mono /home/BDInfoCli.exe /home/cdrom /home

这个命令用到3个地址，第一个是BDInfoCli.exe的地址，第二个是蓝光文件夹的地址，第三个是输出info的地址
扫描开始，须选择正确的序号
扫描中。。。。。。
结束查看文件

ls /home/

其中BDINFO.cdrom.txt就是我们需要的
可以传到本地，也可以在盒子里看

nano /home/BDINFO.cdrom.txt

得到BDinfo后，由于有些站点不允许发布ISO文件，还要把文件提取出来

cp –rf /home/cdrom/* /home/transmission/downloads/The.Rosary.Murders.1987.COMPLETE.BLURAY-watchHD/

这个命令是/home/cdrom/目录下的所有文件copy到后一个文件夹（前一个是挂载点，后一个可以自己定义）
查看文件

ls

只有蓝光文件夹了，可以制作种子了。
对于一般重编码视频，需要获取mediainfo
安装mediainfo

apt-get install mediainfo

以/home/xxx为例

mediainfo /home/xxx > /home/e.txt
nano /home/e.txt

制作种子
安装mktorrent

apt-get install mktorrent
mktorrent -v -p -l 21 -a http://tracker/announce.php ‘/home/xxx’ -o /home/xxx.torrent

最后几个参数分别是tracker地址，蓝光圆盘地址，种子名字
有的站点有截图要求，截图可以用ffmpeg
安装ffmpeg

apt-get install ffmpeg

由于ubuntu软件仓库里的ffmpeg版本较旧,截图可能出现问题,推荐安装最新的ffmpeg
切换到视频文件夹，以蓝光为例

cd /home/xxx/BDMV/STREAM

截图

ffmpeg -ss 00:10:23 -i “00004.m2ts” -vframes 1 -y -f image2 “/home/transmission/downloads/screenshot1.png”

通过修改时间和名字，可以获得多个截图。

通过以上步骤，得到了种子，视频信息和截图，只要拖到本地就可以发布，然后用盒子校验了。
